# 🏦 Bank Management System (Python OOP)

Hi 👋  
I'm **Shadab Alam**, and this project is a simple yet practical Bank Management System built using Python and Object-Oriented Programming (OOP).

This project simulates real-world banking operations like account management, transactions, and data handling.

---

## 📌 Project Overview

This is a menu-driven CLI (Command Line Interface) application where users can perform basic banking operations such as:

- Creating a bank account  
- Depositing and withdrawing money  
- Updating user details  
- Deleting an account  

The goal is to understand how backend logic of a banking system works.

---

## 🚀 Features

- Create Account  
- View Account Details  
- Update Account Information  
- Delete Account  
- Deposit Money  
- Withdraw Money  
- Check Account Balance  

---

## 🧠 Concepts & Tech Used

### 🔹 Object-Oriented Programming (OOP)
- Classes & Objects  
- Constructors (`__init__`)  
- Instance variables & methods  

### 🔹 Core Programming
- Conditional Statements (if-else)  
- Loops (menu-driven system)  
- Functions  

### 🔹 Data Handling
- JSON file (`data.json`) for data persistence  
- CRUD operations (Create, Read, Update, Delete)  

---

## ⚙️ How It Works

- Each bank account is represented as an object  
- All accounts are stored in a JSON file  
- On program start:
  - Data is loaded from `data.json`  
- User interacts via a menu system  
- Changes are saved back to the file automatically
