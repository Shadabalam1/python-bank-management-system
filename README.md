# 🏦 Bank Management System (Python OOP)

Hi 👋
I'm **Shadab Alam**, and this project is a simple Bank Management System built using Python and Object-Oriented Programming concepts.

The goal of this project is to understand how real-world systems work by implementing core banking operations like creating accounts, managing balance, and updating user data.

---

## 📌 About the Project

This is a menu-driven Python application where users can perform basic banking operations.

I built this project to practice:

* OOP concepts
* Logic building
* Real-world problem solving

---

## 🚀 Features

* Create Account
* Show Account Details
* Update Account Information
* Delete Account
* Deposit Money
* Withdraw Money
* Check Balance

---

## 🧠 Concepts Used

### 🔹 Object-Oriented Programming (OOP)

* Classes & Objects
* Constructors (`__init__`)
* Instance variables
* Methods

### 🔹 Core Programming

* Conditional statements (if-else)
* Loops (while loop for menu system)
* Functions

### 🔹 Data Handling

* Dictionary used to store multiple accounts
* Basic data management (CRUD operations)

---

## ⚙️ How It Works

* Each user account is created as an object
* All accounts are stored in a dictionary
* User interacts with the system using a menu
* Based on user input, different operations are performed
